# FixMyLife AI

An AI-powered everyday problem-solving app for normal people. Your first stop when you don't know what to do.

## 🎯 What It Does

FixMyLife AI helps users solve daily life problems across multiple domains:

- 💼 **Jobs & Career** - Resume building, interview prep, salary negotiation
- 💰 **Money & Finance** - Budget planning, investment advice, financial literacy
- 📚 **Education & Skills** - Learning paths, scholarships, skill development
- 🏥 **Health Basics** - Medical report explanations, wellness guidance
- ⚖️ **Legal & Government** - Form filling, bureaucracy navigation, PAN card help
- 📝 **Documents** - Letter writing, application templates, official documents

## 📱 Tech Stack

### Frontend
- **Framework**: React Native / Flutter (cross-platform)
- **State Management**: Redux / Riverpod
- **Voice**: React-Native-Voice / Google Speech API

### Backend
- **Runtime**: Node.js (Express)
- **Authentication**: JWT + OAuth2
- **Database**: PostgreSQL
- **Cache**: Redis
- **AI Integration**: OpenAI API

### Infrastructure
- **Hosting**: AWS / GCP
- **Storage**: S3 / Cloud Storage
- **CI/CD**: GitHub Actions
- **Containerization**: Docker

## 🔧 Getting Started

### Prerequisites
- Node.js 16+
- PostgreSQL 12+
- Docker (optional)
- OpenAI API key

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```
